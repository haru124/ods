1. Create git repo with readme, license and gitignore
2. edit readme to include description
3. add ml specific section to gitignore
4. clone repo to local machine
5. create and switch to new branch dev
6. create venv
python -m venv odslib

7.activate it — do this EVERY time you open a new terminal
odslib\Scripts\activate

8. install packages one group at a time

pip install torch torchvision torchaudio

pip install ipykernel ipywidgets
