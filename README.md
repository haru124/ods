# ODS — Object Detection and Segmentation

A project on object detection and semantic/instance segmentation focused on autonomous driving applications.

## Overview
- Detect vehicles, pedestrians, and road objects using bounding boxes
- Segment road scenes at pixel level (road, sky, building, car, person, etc.)
- Dataset: Cityscapes (primary), BDD100K** (generalization testing)
- Framework: PyTorch
